Extração Automática de Figuras de Mérito

Projeto para análise automática de transistores MOS utilizando Python

Sobre o Projeto

Este projeto utiliza algoritmos Python para a extração de parâmetros como mobilidade, tensão de limiar e outros em transistores MOS. A interface desta página exibe os resultados das análises.
